  ### Bezier: An SVG editing web app

  This app is an Illustrator-style SVG path editor built to create and refine vector artwork directly in the browser. 
  It supports multi-path editing with anchor points and bezier handles, transform tools (move/scale/rotate), zoom and pan controls, 
  and a live SVG code pane so visual edits and raw markup stay in sync for fast iteration and export.

  <ContentImage1Column>
    <ImageContainer>
      <Image
        src={img1}
        alt="SVG editor"
        width={2816}
        loading="lazy"
        quality={100}
      />
    </ImageContainer>
  </ContentImage1Column>

  It’s built with React + TypeScript on Vite, with a component-based UI and dedicated geometry utilities for path operations. 
  Quality is backed by a BDD-style testing approach, using Vitest and Playwright to validate expected user behaviors and key editing workflows end to end.
